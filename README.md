### Pre-requisites

- Python 2.7.x (tested on Python 2.7.11)
- [sshtunnel](https://pypi.python.org/pypi/sshtunnel) module

### Usage

```bash
> python src/extractd.py --host <hostIP> --username <username> --password <password>
```
